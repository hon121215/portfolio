# 송정훈 포트폴리오

AI, 데이터, 컴퓨터비전, 지능형 교통 시스템 중심으로 수행한 연구, 논문, 특허, 수상, 자격증을 정리한 포트폴리오입니다.

## Overview

- 관심 분야: Computer Vision, Out-of-Distribution Detection, AI for Mobility, Data Analysis
- 핵심 키워드: 자율주행, 도로 객체 탐지, 이상 상황 해석, 태양광 발전 예측, 음성 변환
- 포트폴리오 구성: 논문 및 연구, 특허 및 아이디어, 수상 실적, 자격증, 대외 활동

## Suggested Narrative

이 포트폴리오는 단순한 결과물 모음이 아니라, 문제를 정의하고 데이터를 다루며 모델을 설계하고 실제 응용 영역에 연결하는 역량을 보여주는 방향으로 구성했습니다.

특히 다음 흐름이 강점으로 보입니다.

1. 컴퓨터비전 및 분포 외 객체 탐지(OoD) 연구
2. 자율주행 및 지능형 교통 시스템 응용
3. 데이터 기반 예측 및 분석 역량
4. 학술 성과와 실전 대회 성과의 동시 보유
