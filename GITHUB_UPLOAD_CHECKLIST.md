# GitHub 업로드 체크리스트

## 공개 전 확인

- 주민등록번호, 생년월일, 주소, 전화번호, 이메일 전체 주소가 PDF에 노출되는지 확인
- 자격증 번호, 수험번호, 발급번호, 서명 이미지가 포함되어 있는지 확인
- 특허 또는 논문 중 외부 공개가 제한된 자료가 있는지 확인
- 발표자료에 학교, 기관, 공동저자 개인정보가 과도하게 포함되어 있는지 확인

## 저장소 생성 순서

1. GitHub에서 새 저장소를 생성
2. 현재 폴더 내용을 저장소에 업로드
3. `README.md`가 메인 화면에 표시되는지 확인
4. PDF 링크가 정상적으로 열리는지 확인
5. 필요하면 저장소를 `Public` 또는 `Private`로 전환

## 추천 공개 전략

- 완전 공개용: 수상, 자격증, 공개 가능한 논문/발표자료만 업로드
- 비공개 면접용: 전체 증빙 자료 업로드
- 혼합 전략: 공개 저장소에는 README와 일부 대표 자료만 두고, 나머지는 비공개 저장소 또는 클라우드 링크로 관리
